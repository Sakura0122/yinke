## async await

用同步方式 执行异步操作

```js
async function fn () {
  await request(1)
  await request(2)
}
fn()
```

- 特点
- await 只能在async函数中使用
- async 返回的是promise
- await 后面是promise
- async/await 用同步方式 执行异步操作

## generator

function* gen () {

}
